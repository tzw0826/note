    eg：

    | 专业代码 | 专业名称 |
    | -------- | -------: |
    | 01       | 市场营销 |
    | 02       | 财务管理 |
    | 03       |   计算机 |
    | 04       | 国际贸易 |

    =lookup(A1,{"01","02","03","04"},{"市场营销","财务管理","计算机","国际贸易"})
